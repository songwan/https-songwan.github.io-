### Required packages & versions
- R version 3.2.0 or higher
- devtools
- roxygen2
- testthat
- knitr

----
### Package structure
#### 2.1 Naming packages
- Consists of letters, numbers, and periods.
- Starts with a letter, not end with a period.
- Not recommend to use periods.
